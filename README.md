# What
Galactic warfare game

# Setup
1. Download Android Studio
2. (Optional?) Download libGDX project setup tool: https://libgdx.com/dev/project-generation/
3. VCS -> Checkout from version control -> Git
URL = https://github.com/hupeter/GalacticWarfare
Directory = <your directory>
4. File -> Sync file with Gradle files
5. Gradle -> GalacticWarfare -> desktop -> Tasks -> other -> run
6. VCS -> Enable version control integartion -> Git
7. Make a change
8. VCS -> Commit
9. VCS -> Git -> Push
